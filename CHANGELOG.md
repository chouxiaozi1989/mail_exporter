# 更新日志

本文档记录了邮件导出工具的所有重要更新和功能变更。

## [v1.5.0] - 2024-12-19

### 🔐 新增功能
- **OAuth2认证支持**：Gmail支持OAuth2安全认证，无需应用密码
- **智能界面优化**：OAuth2认证时自动隐藏/显示相关字段
- **重新授权功能**：支持OAuth2重新授权，方便切换账户
- **增强状态显示**：详细的OAuth2认证状态和操作提示

### 🎨 界面改进
- OAuth2认证字段智能显示/隐藏
- 增强的状态提示信息，包含详细的操作指导
- 优化的用户体验，减少界面混乱

### 🛡️ 安全性提升
- 使用OAuth2标准协议，提高账户安全性
- 支持PKCE (Proof Key for Code Exchange) 增强安全性
- 敏感文件自动添加到.gitignore

### 📝 文档完善
- 新增详细的OAuth2设置指南 (OAUTH_SETUP_GUIDE.md)
- 更新README.md，添加OAuth2功能说明
- 完善安装和使用说明

### 🔧 代码优化
- 清理临时和测试文件
- 优化requirements.txt依赖列表
- 统一代码风格和结构

## [v1.4.0] - 2024-11-15

### 🌐 新增功能
- **系统代理支持**：解决Gmail连接问题
- **代理连接优化**：支持HTTP代理和直连回退

### 🔧 改进
- 改进Gmail IMAP连接稳定性
- 增强网络连接错误处理机制

## [v1.3.0] - 2024-10-20

### 🏢 新增功能
- **多邮箱服务商支持**：支持Gmail、QQ邮箱、163邮箱、126邮箱、新浪邮箱等
- **增量保存功能**：支持按条写入邮件数据，点击停止时保存已处理的邮件
- **附件下载支持**：新增邮件附件下载功能
- **JSON格式支持**：除CSV外，新增JSON格式导出选项

### 🔧 改进
- 邮箱服务商自动识别：根据邮箱地址自动配置IMAP服务器设置
- 停止按钮优化：将"取消"按钮改为"停止"按钮，提供更好的用户体验
- 错误处理优化：修复"无法确定邮箱服务提供商配置"等错误

## [v1.2.0] - 2024-09-10

### ✨ 新增功能
- **图形界面支持**：提供完整的GUI界面
- **实时进度显示**：显示邮件处理进度和状态信息

### 🐛 修复
- 修复CSV文件只写入最后一批数据的问题
- 改进进度显示和错误处理

### 🛡️ 稳定性

## 版本说明

### 版本号格式
本项目使用语义化版本号 (Semantic Versioning)：
- **主版本号**：不兼容的API修改
- **次版本号**：向下兼容的功能性新增
- **修订号**：向下兼容的问题修正
